## Theme Definitions

Once the user selects a theme, apply the corresponding style rules below for ALL frontend code generation in the session.

### Theme 1 — Neubrutalism
- **Accent**: `#FF5C00`
- **Tags**: raw, bold-shadows, exposed
- **Direction**: Bold saturated colors, thick black borders, hard offset drop shadows, zero rounded corners, raw structural energy with playful confidence. Everything boxed, everything unapologetic. Cards feel stamped onto the page.

### Theme 2 — Kinetic Typography
- **Accent**: `#1A1A1A`
- **Tags**: typography, kinetic, oversized
- **Direction**: TEXT IS THE ENTIRE DESIGN. No icons, no illustrations — only letters and numbers. Metrics as GIANT numerals (200px+). Words overlap, stack, intersect. Mix ultra-bold condensed and ultra-thin extended weights. Monochrome with ONE color accent on one word. David Carson's Ray Gun magazine as a dashboard.

### Theme 3 — Sketchy Wireframe
- **Accent**: `#9CA3AF`
- **Tags**: wireframe, sketch, minimal, lo-fi
- **Direction**: Hand-drawn aesthetic — ALL lines squiggly via SVG filters (feTurbulence + feDisplacementMap) or irregular border-radius. Thin wobbly borders in gray. Handwriting fonts ('Architects Daughter', 'Patrick Hand', 'Caveat'). Placeholder-style content blocks, cross-hatching for fills. NO color beyond grays and white.

### Theme 4 — Aurora UI
- **Accent**: `#6366F1`
- **Tags**: mesh-gradient, luminous, modern
- **Direction**: Dark base with ethereal gradient mesh backgrounds, northern-lights color flows using layered radial gradients, glassmorphism cards with backdrop-blur, soft luminous depth. Background feels alive and atmospheric.

### Theme 5 — Bento Grid
- **Accent**: `#0A84FF`
- **Tags**: bento, modular, oversized
- **Direction**: Japanese bento-box modular layout with genuinely asymmetric tiles of varying sizes (2×2, 1×1, 1×2), oversized typography inside tiles, clean compartmentalized sections. The grid itself is the design statement.

### Theme 6 — Anti-Design
- **Accent**: `#FF0040`
- **Tags**: chaotic, punk, rule-breaking
- **Direction**: Intentionally clashing colors, broken grid layouts, wildly mixed typefaces and sizes, rotated elements, overlapping content. Dissonance and tension are the point. Rules exist to be violated with attitude.

### Theme 7 — Dark Precision SaaS
- **Accent**: `#A78BFA`
- **Tags**: dark, precise, saas
- **Direction**: Near-black backgrounds, ultra-subtle borders, muted accent used sparingly, razor-sharp compact typography. Linear/Vercel/Raycast aesthetic. Precision engineering made visible through restrained elegance. Compact, information-dense, nothing wasted.

### Theme 8 — Dark Maximalism
- **Accent**: `#F59E0B`
- **Tags**: dark, rich, maximalist
- **Direction**: Rich dark palette with warm undertones, layered surfaces with visible depth, gold and jewel-tone accents, ornate decorative details, noise/grain textures, dramatic typography mixing serif display and sans-serif body. Baroque ambition meets dark mode.

### Theme 9 — Liquid Motion
- **Accent**: `#8B5CF6`
- **Tags**: fluid, organic, animated
- **Direction**: Fluid morphing blob shapes as backgrounds, organic curves everywhere, pill-shaped buttons, large rounded corners, elastic hover animations, gradient fills that shift on interaction. The UI feels like a living organism — nothing static, everything breathes.

### Theme 10 — Monochrome Luxe
- **Accent**: `#1A1A2E`
- **Tags**: monochrome, editorial, luxe
- **Direction**: Strictly black, white, and grays with one deep accent used sparingly. All visual interest from typography and space. Rich serif display headings, clean sans-serif body. Generous whitespace, editorial column structure, hairline rules. Constraint breeds elegance.

### Theme 11 — Cyberpunk HUD
- **Accent**: `#00FFFF`
- **Tags**: hud, glitch, tactical
- **Direction**: Very dark background, neon accents (cyan, magenta), monospace typography, dense tactical layout, scanline overlays, glitch animations, corner-bracket framing, neon glow on borders and text. Sci-fi dashboard through augmented-reality implants.

### Theme 12 — Organic Minimalism
- **Accent**: `#5B7553`
- **Tags**: earth, organic, nature
- **Direction**: Warm earth tones (olive, clay, sage, stone), generous whitespace, flowing organic composition, large rounded corners, soft warm shadows. Hand-drawn SVG illustrations. Nature-inspired curves replace geometric rigidity. Ceramics studio confidence.

### Theme 13 — Collage Punk
- **Accent**: `#EF4444`
- **Tags**: mixed-media, zine, lo-fi
- **Direction**: Zine aesthetic — mixed typefaces (typewriter, handwritten, bold sans), cards at slight rotations, tape-and-sticker elements, torn paper edges, newsprint textures, stamp-like badges. Lo-fi and tactile, assembled with scissors and glue.

### Theme 14 — Swiss Clean
- **Accent**: `#FF0000`
- **Tags**: grid, helvetica, clean
- **Direction**: Strict mathematical grid, black and white with one red accent, left-aligned everything, clear typographic hierarchy through size and weight alone, zero decorative elements, generous whitespace. The grid IS the decoration.

### Theme 15 — High-Contrast Duotone
- **Accent**: `#FFD600`
- **Tags**: accessible, duotone, stark
- **Direction**: Strictly two base colors (black and white) with one bright accent. No mid-tones. Large legible typography, thick borders, oversized touch targets, bold foreground/background separation. Accessibility as aesthetic statement. Stark, surgical, impossible to misread.

### Theme 16 — Grain & Texture Analog
- **Accent**: `#B45309`
- **Tags**: grain, analog, vintage
- **Direction**: Visible noise/grain overlays on every surface, muted vintage tones (rust, ochre, olive, slate), warm off-white backgrounds like aged paper, slightly desaturated faded colors. Film-stock warmth and tactile depth. Classic serif/sans with typewriter accents.

### Theme 17 — Blueprint Wireframe
- **Accent**: `#3B82F6`
- **Tags**: blueprint, technical, exposed
- **Direction**: Visible graph-paper grid lines as primary structure, monospace annotation labels, measurement lines and dimension callouts, crosshair markers, corner coordinates. The interface shows its own construction like a technical drawing.

### Theme 18 — Editorial Serif
- **Accent**: `#B91C1C`
- **Tags**: serif, editorial, magazine
- **Direction**: Sophisticated serif typography (Playfair Display or DM Serif Display) for ALL headings, dramatic size contrast (72px+ headlines, 14px body), generous whitespace (40-60px+ padding), thin 1px rule separators, pull-quote callouts, asymmetric magazine columns, black-and-cream with deep red accent. Vogue's art director designed your dashboard.

### Theme 19 — MySpace / Geocities
- **Accent**: `#FF00FF`
- **Tags**: retro-web, chaotic, maximalist, nostalgic
- **Direction**: Late-90s personal homepage chaos. Tiled backgrounds, hot pink + electric blue + lime green, Comic Sans mixed with Times New Roman, fake visitor counters, marquee-style scrolling (CSS animation), beveled borders, sparkle decorations, "under construction" energy. Maximum self-expression, zero design system, pure creative joy.

### Theme 20 — shadcn Default
- **Accent**: `#18181B`
- **Tags**: clean, minimal, saas, modern
- **Direction**: shadcn/ui default theme. White backgrounds, zinc-50 (#FAFAFA) surfaces, zinc-200 (#E4E4E7) borders (1px solid). Inter or Geist Sans, 14px base. Small border-radius (6px buttons, 8px cards). Primary button: solid black with white text. Cards with thin border + shadow-sm. Consistent 16px/24px spacing. Zero decorative elements — spacing and consistency IS the design.

## Global Design Rules (Apply to ALL Themes)

These rules apply regardless of which theme is selected:

### Production-Grade Code
- Complete, self-contained HTML with ZERO external dependencies (no CDN scripts)
- ALL styles in a single `<style>` block using plain CSS
- CSS custom properties (`--color-*`, `--font-*`, `--radius-*`) for theming
- All icons as inline SVG

### Responsive Design
- Include `<meta name="viewport" content="width=device-width, initial-scale=1">`
- Mobile-first with CSS media queries to scale up
- At 375px: stack sidebars vertically, single-column grids, proportional typography, no horizontal overflow
- `html` and `body` at `width: 100%` with no horizontal margin/padding
- Sidebar layouts: CSS grid/flex where sidebar + main span 100% viewport

### Typography
- NEVER use generic fonts (Arial, Inter, Roboto, system fonts) unless the theme explicitly specifies them
- Choose distinctive, characterful fonts that match the theme
- Pair display fonts with refined body fonts

### Visual Quality
- Dominant colors with sharp accents — avoid timid, evenly-distributed palettes
- CSS-only animations preferred for HTML output
- Unexpected layouts: asymmetry, overlap, diagonal flow, grid-breaking elements
- Atmosphere and depth over solid color backgrounds

### Anti-Patterns (NEVER Do)
- Generic "AI slop" aesthetics
- Purple gradients on white backgrounds (unless theme-specific)
- Predictable layouts and cookie-cutter components
- Converging on the same fonts across different themes
